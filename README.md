# رحيق - متجر العسل الطبيعي

موقع إلكتروني متكامل لبيع العسل الطبيعي مع لوحة تحكم إدارية.

## التقنيات المستخدمة

### Frontend
- React.js 18
- Vite
- Tailwind CSS
- React Router
- Framer Motion
- Axios
- React Hook Form
- Chart.js

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT
- Multer (لرفع الصور)
- WhatsApp Business API (لإرسال رموز التحقق)

## هيكلية المشروع

```
honey-store/
├── frontend/          # React Application
├── backend/           # Node.js/Express API
└── README.md
```

## الإعداد والتشغيل

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
# Create .env file with required variables
npm run dev
```

## المتغيرات البيئية المطلوبة

انظر إلى `.env.example` في مجلد `backend` للمتغيرات المطلوبة.
